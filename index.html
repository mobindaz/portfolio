Vote Checker

The Vote Checker is a web application that verifies voter eligibility based on age. Users can input their age, and the system dynamically determines whether they are eligible to vote. If the user is below 18, a message is displayed indicating their ineligibility; otherwise, they are informed that they are eligible to vote.

The project is built using HTML, CSS, and JavaScript, providing a simple and user-friendly interface. It serves as an educational tool to raise awareness about voting eligibility requirements.
